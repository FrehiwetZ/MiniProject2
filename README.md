# Personal Profile Page (Mini Project)

##  Project Description
This is a simple **Personal Profile Web Page** created as a mini project using **HTML and CSS only**.  
The project is designed to practice core web development concepts such as layout design, styling, and responsive design without using any CSS frameworks.

---

##  Project Goals
- Practice proper HTML structure
- Apply core CSS styling techniques
- Understand Flexbox and Grid layouts
- Build a responsive webpage
- Improve frontend design skills

---

## Technologies Used
- HTML5
- CSS3

 No CSS frameworks (Bootstrap, Tailwind, etc.) were used.

---

##  Project Features
- **Header Section** displaying name and title
- **Profile Section** with image and short bio using Flexbox
- **Skills Section** arranged using CSS Grid
- **Contact Form** with name, email, and message fields
- **Footer Section** with copyright text
- Fully **responsive design** using media queries

---

##  CSS Concepts Applied
- Display properties (`block`, `inline`, `inline-block`)
- Flexbox (`display: flex`, `justify-content`, `align-items`)
- Grid (`display: grid`, `grid-template-columns`, `gap`)
- Text styling (`font-size`, `text-align`, `text-transform`)
- Font styling (`font-family`, `font-weight`)
- Background colors and gradients
- Responsive design using media queries

---

##  Responsive Design
- Layout adjusts for smaller screens
- Flexbox sections stack vertically on mobile devices
- Grid layout switches to a single column
- Images resize properly for different screen sizes

---

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. The webpage will load locally
